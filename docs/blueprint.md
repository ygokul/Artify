# **App Name**: Artify Canvas

## Core Features:

- Drawing Canvas: Clean and intuitive drawing canvas with essential tools like brushes, erasers, color palettes, and undo/redo functionalities.
- AI Image Generation: AI Image Tool: type a text prompt and use the tool to generate an image. Once generated, the result can be saved or further edited in the Drawing Canvas.
- Artwork Display: A dedicated space for users to view, manage, and showcase their created artworks.
- Admin Contests: Allow the admin to generate contests with a defined amount of time, then display the best artist submissions

## Style Guidelines:

- Primary color: HSL(210, 70%, 50%) – a vibrant blue (#3399FF) that conveys creativity and innovation.
- Background color: HSL(210, 20%, 95%) – a very light blue (#F0F8FF) providing a subtle, clean backdrop that does not distract from the artwork.
- Accent color: HSL(180, 60%, 55%) – a teal color (#33D9FF) that complements the primary blue, drawing attention to interactive elements and calls to action.
- Font pairing: 'Poppins' (sans-serif) for headlines, providing a contemporary and fashionable look; 'PT Sans' (sans-serif) for body text, offering readability and a modern feel.
- Use minimalist, outline-style icons for key functions, to provide a modern and user-friendly interface. Prefer the use of black or the primary color, on a light background.
- Employ a grid-based layout for both the drawing canvas interface and the artwork display. Keep the layout clean and mobile-first. Prioritize ease of navigation and quick access to tools and galleries.
- Use subtle transitions for tool selections and gallery browsing. On artwork loading, use a soft fade-in effect to enhance the visual experience without being distracting.
